注意：

  • const void *a定义一个指针a,a可以指向任意类型的值，且它指向的值必须是常量。在这种情况下，我们不能修改被指向的对象，但可以使指针指向其他对象。
  • void* const a定义了一个const指针a，a可以指向任意类型的值，但a是指向某个对象的常量指针。我们不能修改指针中存储的地址，但可以修改指针指向的对象。
